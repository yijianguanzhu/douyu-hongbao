#### ![配置](https://www.easyicon.net/api/resizeApi.php?id=1183257&size=24) 自定义配置
    在/core/config.ini文件中，可自定义特定参数（首次运行会自动生成）:
        [monitor-thread]
          # 每隔多少秒检测一次红包（默认：10秒）
		  period=10
		[handler-thread]
		  # 抢红包线程启动偏移量，取值范围：[-2.00,2.00]（单位秒）（保留两位小数）（默认：0）
		  leftshift=0
		[broker-thread]
		  # 小礼物是否自动赠送（大气、666和办卡）（默认：true）
		  small=true
          # 大礼物是否自动赠送（飞机、火箭）（默认：false）
		  precious=false
